# <img src="https://github.com/user-attachments/assets/86c2f469-5071-49af-ac1f-12a657989df9" width="40"/> Rakshak DTU

RakshakDTU is an AI-powered vehicle monitoring system that utilizes CCTV cameras installed at various location in campus like at entry and exit points to detect vehicle number plates using technologies such as YOLO and Optical Character Recognition (OCR). The system cross-references detected vehicles with a registered database and logs entry and exit events in real time, thereby enabling efficient monitoring and improved security.

## 🚀 Features

### Core Monitoring
- Real-time Entry/Exit Logs: Automated logging of vehicles entering and exiting campus gates.
- Smart Filtering: Dedicated views for vehicles currently On Campus, All Logs, and Unauthorized entries.
- Camera Administration: A management interface to view, add, and monitor the status of surveillance cameras across the premises.

### Vehicle Management
- Centralized Registry: Complete CRUD (Create, Read, Update, Delete) functionality for registered vehicles.
- Search & Filter: Instantly find vehicles by License Plate, Owner Name, or Department.

### Modern UI/UX
- Animated Splash Screen: Professional, branded entry experience.
- Dark Mode Support: Optimized for 24/7 security monitoring in all lighting conditions.
- Clean Material Design: Intuitive navigation for security personnel and administrators.
